# Image Recognition Model
## 1. Model Training
I created an image recognition model using Google Teachable Machine.
The model contains two classes:
- Class 1: Dog
- Class 2: Cat
I uploaded images for each class and trained the model.

## 2. Model Evaluation
After training the model, I tested it using new images to evaluate the prediction results.

## 3. Model Export
The trained model was exported from Teachable Machine using:
TensorFlow → Keras
The exported files are:
- keras_model.h5
- labels.txt

## 4. Python Implementation
A Python script was created to:
- Load the trained Keras model.
- Accept an input image.
- Predict the image class.

## 5. Output
The Python script successfully predicted the input image class.
The screenshot of the output is included in this repository.
